Waarom ben ik begonnen met programmeren?

Ik ben 3 jaar geleden begonnen met programmeren toen ik ontdekte hoe je plugins voor Minecraft kon maken. Ik vond het leuk dat ik zelf programma's kon maken om het spel naar mijn wensen te veranderen. Ik begon met een plugin voor Minecraft waar ik nog steeds mee bezig ben. Een jaar geleden ben ik geslaagd voor mijn VMBO-TL diploma en ben ik daarna begonnen met mijn opleiding Software Developer.

Wat zijn mijn doelen?

Ik wil na mijn opleiding vloeiend willen zijn in Java Development. Java is op dit moment erg gewild in de Developer wereld. Het is mijn droom om in de toekomst te mogen werken als developer bij de Efteling, omdat ik een grote fan ben van de Efteling.
