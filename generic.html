Mij

Mijn naam is Mees. Ik ben 18 jaar oud en ik woon in Heerhugowaard. Mijn hobby's zijn gamen, programmeren (Liefst in Java/C#) en de Efteling. Ik werk bij de Koninklijke Marine als ICT'er.

Waarom ben ik begonnen met programmeren?

Ik ben 5 jaar geleden begonnen met programmeren toen ik ontdekte hoe je plugins voor Minecraft kon maken. Ik vond het leuk dat ik zelf programma's kon maken om het spel naar mijn wensen te veranderen. Ik begon met een plugin voor Minecraft waar ik nog steeds mee bezig ben. 2 jaar geleden ben ik geslaagd voor mijn VMBO-TL diploma en ben ik daarna begonnen met mijn opleiding Software Developer.

Wat zijn mijn doelen?

Ik wil na mijn opleiding vloeiend willen zijn in Java Development. Java is op dit moment erg gewild in de developer wereld. Het is mijn droom om in de toekomst te mogen werken als developer bij de Efteling, omdat ik een grote fan ben van de Efteling.
